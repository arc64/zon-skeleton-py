# Prerequisites

* Python 2.6.1
* Requests
* JSON

## Install requests (http://docs.python-requests.org/en/latest/)
	pip install requests


# How to run with it

1. Clone this repo
2. Get your key: http://developer.zeit.de/quickstart/
3. Put it into the example.py file
4. From the console run this:

        ./example.py

Go hack...
