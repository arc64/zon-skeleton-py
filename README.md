# Prerequisites

* Python 2.6.1
* Requests
* JSON

## Install requests (http://docs.python-requests.org/en/latest/)
	pip install requests


## API key
	Get one here: http://developer.zeit.de/quickstart/

# Run it
	Clone this repo
	Get your key and put it into the example file
	
	From the console run this:
	./example.py
